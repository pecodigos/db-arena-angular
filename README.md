# DB-Arena Angular frontend

Angular frontend for Dragon Ball Arena.

## Install

```bash
npm install
```

## Run locally

```bash
npm start
```

Frontend URL:
- http://localhost:4200

## Build

```bash
npm run build
```

## Backend connection

Environment files live in:
- `src/environments/environment.ts`
- `src/environments/environment.development.ts`
- `src/environments/environment.prod.ts`

Default backend URL:
- `http://localhost:8080`
